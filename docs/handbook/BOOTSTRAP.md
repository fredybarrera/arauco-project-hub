# BOOTSTRAP

## Bienvenido a Arauco Project Hub

Si eres un nuevo integrante del proyecto (persona o asistente de IA), este es el primer documento que debes leer.

El propósito de este documento es permitir que cualquier colaborador comprenda el proyecto y pueda incorporarse al trabajo en pocos minutos.

---

# ¿Qué es Arauco Project Hub?

Arauco Project Hub es una plataforma corporativa diseñada para administrar el ciclo de vida completo de las iniciativas de desarrollo de software dentro de Arauco.

Su objetivo es centralizar toda la información relacionada con una iniciativa, desde su origen hasta la operación del software en producción.

No es un sistema de tickets.

No es un gestor de proyectos tradicional.

Es una plataforma para administrar iniciativas tecnológicas.

---

# Principios del Proyecto

Antes de modificar cualquier documento o escribir código, debes comprender los siguientes principios:

* El negocio define el software.
* La Iniciativa es el Aggregate Root del dominio.
* El Lenguaje Ubicuo es obligatorio.
* La documentación forma parte del producto.
* La simplicidad tiene prioridad sobre la configurabilidad.
* Toda decisión importante debe documentarse.
* El repositorio constituye la memoria oficial del proyecto.

# Cómo colaborar

Antes de realizar cualquier cambio:

1. Comprender el problema.
2. Revisar si existe un documento relacionado.
3. Mantener coherencia con el dominio.
4. No crear nuevos conceptos sin justificación.
5. Documentar toda decisión importante.

---

# Objetivo del Engineering Playbook

El Engineering Playbook representa la memoria viva del proyecto.

Toda decisión arquitectónica, funcional o técnica debe quedar registrada aquí antes de ser implementada en el software.
